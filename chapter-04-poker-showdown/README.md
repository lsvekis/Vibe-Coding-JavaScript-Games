# Chapter 04 — Poker Showdown

This folder contains the playable code for **Poker Showdown**, aligned to the book chapter of the same name.

## Core concepts
- sorting arrays
- pattern detection
- ranking logic
- multi-step flows

## Why this chapter matters
This chapter moves from simple state to algorithmic thinking through hand evaluation.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `chapter-guide.md`

## Teaching goal
Use this game to understand not just how to write code, but how to reason about systems:
- what the player can do
- what the game must remember
- what rules transform input into outcomes
- how the screen reflects changing state
