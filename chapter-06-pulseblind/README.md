# Chapter 06 — Pulseblind

This folder contains the playable code for **Pulseblind**, aligned to the book chapter of the same name.

## Core concepts
- canvas rendering
- keyboard input
- visibility systems
- spatial gameplay

## Why this chapter matters
This chapter shifts from DOM-heavy UI into canvas-based rendering and player movement.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `chapter-guide.md`

## Teaching goal
Use this game to understand not just how to write code, but how to reason about systems:
- what the player can do
- what the game must remember
- what rules transform input into outcomes
- how the screen reflects changing state
