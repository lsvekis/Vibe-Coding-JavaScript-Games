# Chapter 14 — Second Shadow Heist

This folder contains the playable code for **Second Shadow Heist**, aligned to the book chapter of the same name.

## Core concepts
- stealth logic
- looped replays
- level constraints
- path-based hazards

## Why this chapter matters
This chapter combines route planning, replay systems, and stealth pressure.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `chapter-guide.md`

## Teaching goal
Use this game to understand not just how to write code, but how to reason about systems:
- what the player can do
- what the game must remember
- what rules transform input into outcomes
- how the screen reflects changing state
