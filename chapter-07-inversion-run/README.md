# Chapter 07 — Inversion Run

This folder contains the playable code for **Inversion Run**, aligned to the book chapter of the same name.

## Core concepts
- gravity
- velocity
- collision detection
- timed obstacle spawning

## Why this chapter matters
This chapter introduces lightweight physics thinking using position, speed, and gravity.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `chapter-guide.md`

## Teaching goal
Use this game to understand not just how to write code, but how to reason about systems:
- what the player can do
- what the game must remember
- what rules transform input into outcomes
- how the screen reflects changing state
