# Chapter 02 — Blackjack Command

This folder contains the playable code for **Blackjack Command**, aligned to the book chapter of the same name.

## Core concepts
- arrays and objects
- game state
- conditional logic
- card value calculation

## Why this chapter matters
This chapter teaches how to model a deck, a hand, and turn-based state transitions.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `chapter-guide.md`

## Teaching goal
Use this game to understand not just how to write code, but how to reason about systems:
- what the player can do
- what the game must remember
- what rules transform input into outcomes
- how the screen reflects changing state
