# Chapter 01 — Slot Machine

This folder contains the playable code for **Slot Machine**, aligned to the book chapter of the same name.

## Core concepts
- DOM manipulation
- randomization
- event handling
- localStorage basics

## Why this chapter matters
This chapter introduces the idea that a game is just state, input, rules, and output.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `chapter-guide.md`

## Teaching goal
Use this game to understand not just how to write code, but how to reason about systems:
- what the player can do
- what the game must remember
- what rules transform input into outcomes
- how the screen reflects changing state
