# Introduction — The Way You Learn Has Changed

Learning to code has changed.

You can now open an AI tool, describe an idea, and receive a working starting point in seconds. That shift is powerful, but it also creates a new problem: getting code is no longer the same as understanding code.

That is why this book exists.

**Vibe Coding JavaScript Games** is not just a project book. It is a system-building book. You will learn JavaScript by building fifteen playable games, but the deeper goal is to help you think like a developer who can design state, logic, interaction, and progression deliberately.

This book uses the Vibe Learning format:

1. Curiosity — understand the problem and the player experience  
2. Build — create a working version  
3. Explain — study how the code works and why it is structured that way  
4. Refine — improve the result with prompts, debugging, and iteration  
5. Extend — add meaningful features  
6. Reflect — capture the reusable ideas

You are not here to memorize syntax. You are here to learn how systems work.

By the end of this book, you will not only have built fifteen games. You will also understand how to use AI-assisted development the right way: to accelerate thinking, not replace it.
