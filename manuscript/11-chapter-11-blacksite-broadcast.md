## Explain

Read the `script.js` file in this chapter folder and identify:
- where the state is stored
- which functions apply the main rules
- how the UI is updated
- where the game begins and ends

Study the `chapter-guide.md` file in the same folder for chapter-specific teaching notes.

```text
Build a haunted station management game where the player balances signal, power, and panic over time.
```

Then refine with follow-up prompts such as:
- “Explain what each major function does.”
- “Refactor this code into smaller reusable helpers.”
- “Add one new mechanic without changing the main structure.”
