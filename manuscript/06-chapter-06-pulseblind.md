Core concepts in this chapter:

- hidden information
- timed perception
- movement in darkness

## Explain

Read the `script.js` file in this chapter folder and identify:
- where the state is stored
- which functions apply the main rules
- how the UI is updated
- where the game begins and ends

Study the `chapter-guide.md` file in the same folder for chapter-specific teaching notes.
