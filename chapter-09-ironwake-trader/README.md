# Chapter 09 — Ironwake Trader

This folder contains the playable code for **Ironwake Trader**, aligned to the book chapter of the same name.

## Core concepts
- economy systems
- dynamic pricing
- inventory management
- travel events

## Why this chapter matters
This chapter introduces simulation design through ports, markets, piracy, and ship upgrades.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `chapter-guide.md`

## Teaching goal
Use this game to understand not just how to write code, but how to reason about systems:
- what the player can do
- what the game must remember
- what rules transform input into outcomes
- how the screen reflects changing state
