# Chapter 15 — Hero Dice: Villain Crisis

This folder contains the playable code for **Hero Dice: Villain Crisis**, aligned to the book chapter of the same name.

## Core concepts
- modular architecture
- dice combat
- hero progression
- campaign persistence

## Why this chapter matters
This capstone chapter brings together systems thinking, progression, UI state, and game balance.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `chapter-guide.md`

## Teaching goal
Use this game to understand not just how to write code, but how to reason about systems:
- what the player can do
- what the game must remember
- what rules transform input into outcomes
- how the screen reflects changing state
