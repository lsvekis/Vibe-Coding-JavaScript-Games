# Chapter 12 — Deadzone Sector

This folder contains the playable code for **Deadzone Sector**, aligned to the book chapter of the same name.

## Core concepts
- real-time loops
- enemy waves
- projectiles
- continuous collision checks

## Why this chapter matters
This chapter introduces an action game loop with enemies, bullets, score, and lives.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `chapter-guide.md`

## Teaching goal
Use this game to understand not just how to write code, but how to reason about systems:
- what the player can do
- what the game must remember
- what rules transform input into outcomes
- how the screen reflects changing state
