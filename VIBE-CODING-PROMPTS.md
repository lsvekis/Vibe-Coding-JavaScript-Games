# Vibe Coding Prompts Appendix

Use these prompts to scaffold, extend, and debug each chapter project.

## Chapter 01 — Slot Machine
### Base build prompt
Create a JavaScript slot machine with three reels, a betting system, a jackpot counter, and localStorage so player progress persists after refresh.

### Extension prompts
- Add difficulty scaling without rewriting the entire game.
- Refactor the code into smaller functions and explain what changed.
- Improve the UI while keeping the same game rules.
- Add localStorage persistence where it makes sense.
- Add comments that explain the purpose of each major function.
